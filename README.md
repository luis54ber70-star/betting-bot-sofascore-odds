# 🎯 Betting Bot - 365Scores Odds

Bot de apuestas deportivas que utiliza **365Scores** como fuente de datos para obtener cuotas y partidos en vivo.

## 📋 Descripción

Este bot escanea partidos en vivo y datos históricos desde 365Scores, analiza las cuotas y identifica oportunidades de apuestas.
